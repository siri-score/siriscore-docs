# siriscore-docs

Documentation site for [SiriScore](https://github.com/siri-score/siriscore) — Bitcoin transaction privacy analysis.

**Live at [docs.siriscore.xyz](https://docs.siriscore.xyz)**

## What's here

This repo contains the full reference docs for the SiriScore library, CLI, and REST API — built with [MkDocs](https://www.mkdocs.org) + [Material theme](https://squidfunk.github.io/mkdocs-material/).

## Preview locally

```bash
pip install -r requirements.txt
mkdocs serve
# open http://localhost:8000
```

## Contributing

Docs PRs are welcome. Each page is a plain Markdown file under `docs/`. Open a PR against `main` and the site deploys automatically via GitHub Actions.
